# Express SocketIO Chat

- Practicando con socketIO con express, vanilla js, vue simple, angular.

## Testing

```sh
# playwrigth
npm run test:e2e -- vanilla-e2e
npm run test:e2e -- vue-simple-e2e
npm run test:e2e
# vitest
npm run test
```

## Build angular

```sh
cd public/angular
ng build
```

## Run

```sh
npm install
npm run dev
```
